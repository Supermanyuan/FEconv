1. Mesh conversion

UNV (Universal file) mesh to VTU (Visualization Toolkit) mesh

./feconv inputmesh.unv outputmesh.vtu

BDF (Nastran) mesh to PF3 (Cedrat Flux) mesh:

./feconv inputmesh.bdf outputmesh.pf3

MPHTXT (Comsol) mesh to MFM (Modulef Formatted Mesh) mesh:

./feconv inputmesh.mphtxt outputmesh.mfm

MSH (Ansys) mesh to UNV (Universal file) mesh:

./feconv inputmesh.msh outputmesh.unv

PF3 (Cedrat) mesh to MSH (Ansys) mesh:

./feconv -r soft -t 1.e-7 inputmesh.pf3 outputmesh.msh

MFM (Modulef Formatted File) mesh to MSH (FreeFem++) mesh:

./feconv -ff inputmesh.mfm outputmesh.msh

MSH (FreeFem++) mesh to MPHTXT (Comsol) mesh:

./feconv -ff inputmesh.msh outputmesh.mphtxt

MSH (GMesh) mesh to MUM (Modulef Unformatted Mesh) mesh:

./feconv -gm inputmesh.msh outputmesh.mum

MESH (FreeFeem++) mesh to MUM (Modulef Unformatted File) mesh:

./feconv -ff inputmesh.msh outputmesh.mum

2. Load external fields

Load MFF field

./feconv.exe -if inputfield.mff inputmesh.mfm outputmesh.unv

Load MUF field

./feconv.exe -if inputfield.muf inputmesh.mum outputmesh.unv

Load IP field

./feconv.exe -if inputfield.ip inputmesh.msh outputmesh.vtu

Load DEX field

./feconv.exe -r soft -t 1.e-7 -if inputfield.dex inputmesh.pf3 outputmesh.vtu
Back to top

2. Save external fields

Save MFF field

./feconv.exe -in filename -of outputfield.mff inputmesh.vtu outputmesh.mff

Save MUF field

./feconv.exe -in filename -of outputfield.muf inputmesh.vtu outputmesh.muf

Save IP field

./feconv.exe -in filename -of outputfield.ip inputmesh.vtu outputmesh.msh

Save DEX field

./feconv.exe -in filename -of outputfield.dex inputmesh.vtu outputmesh.pf3
Back to top

2. Change field names

A single field

./feconv.exe -in infilename -on outfilename inputmesh.vtu outputmesh.unv

Several fields

./feconv.exe -in [fn1,fn2,fn3] -of [ofn1,ofn2,ofn3] inputmesh.vtu outputmesh.muf
